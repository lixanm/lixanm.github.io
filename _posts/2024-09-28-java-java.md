## 1.3.java分类

JavaSE:做桌面应用

JavaME:做移动开发或嵌入式开发，已经凉了

JavaEE:做服务器开发



JDK：java开发工具包

​	1.JVM：java虚拟机，真正运行java程序的地方

​	2.核心类库：java提前写好的东西

​	3.开发工具：

​		javac：编译工具

​		java：运行工具

​		jdb：调试工具

​		jhat：内存分析工具

​		......

JRE：java运行环境

​	1.JVM

​	2.核心类库

​	3.jdb：运行工具
