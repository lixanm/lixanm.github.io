## 字符串格式化

“pythin%s”%a

````python

print(b)
//输出：1231

'''
%s   占位字符串
%d   占位整数
%f   占位浮点数
'''

name='理性暗面'
year=2024
price=19.99
c="我是%s,今年是%d,%f" % (name,year,price)
print(c)
//输出:我是理性暗面,今年是2024,19.990000
````
